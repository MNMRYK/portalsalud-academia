Portal de Salud Integrativa (Demo)
Este repositorio contiene la plantilla base de un portal de salud integrativa, diseñado y desarrollado por MN Design Web. Es una solución escalable orientada a clínicas y profesionales de la salud que buscan digitalizar la gestión de pacientes y la oferta de recursos formativos.

🚀 Características principales
Gestión de Pacientes: Panel intuitivo para el seguimiento y administración de fichas clínicas.

Academia Online: Sistema integrado para la gestión de cursos, lecciones y material educativo para pacientes.

Biblioteca de Recursos: Acceso centralizado a recursos clínicos y documentos.

UI/UX Profesional: Diseño limpio y funcional enfocado en la experiencia del usuario final.

🛠 Stack Tecnológico
Este proyecto ha sido construido utilizando tecnologías modernas para garantizar rendimiento y escalabilidad:

Framework: React con TanStack Start.

Estilos: CSS modular y componentes de interfaz (UI).

Herramientas: Vite, TypeScript.

📋 Sobre el proyecto
Este portal sirve como Plantilla Base (Boilerplate) para nuevos proyectos. Su estructura modular permite personalizar la identidad de marca, los flujos de trabajo clínicos y las funcionalidades específicas según las necesidades de cada cliente, manteniendo siempre una base sólida y probada.
